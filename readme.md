# StatsTagger — DonutSMP Minecraft Mod

StatsTagger is a lightweight Minecraft mod made specifically for **DonutSMP** that displays **money stats and kills stats**.

---

## What It Does

Statgger displays **DonutSMP player stats in nametag**

It curently shows:

* Player kills
* Player money

All data is fetched using a DonutSMP stats API and rendered cleanly on-screen.

## Instaltion

1. Download the latest **StatsTagger** build
2. Drop it into your Minecraft `mods` folder
3. Launch Minecraft with fabric api
4. Join **DonutSMP** and you’re good
